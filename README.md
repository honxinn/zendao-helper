## 🚀 快速开始

### 📦 安装步骤
1. **🔧 安装扩展**
   - 安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)
   
2. **📝 创建脚本**
   - 打开扩展管理页面
   - 点击"新建脚本"
   - 复制 `zendao.js` 的全部内容
   
3. **💾 保存启用**
   - 按 `Ctrl+S` 保存脚本
   - 确保脚本已启用
   
4. **🎯 开始使用**
   - 访问禅道系统
   - 首次使用输入用户名
   - 享受高效工作体验！
